# Nightly Dream: Idea Consolidation & Prioritization

## Title
Consolidate and Prioritize Nightly Dream Ideas

## Description
The `misc/nightly-dreams` directory currently contains a large number of idea files, many of which are variants of similar themes (e.g., multiple "consolidation", "curation", "selection", "deduplication" files with different timestamps). This proliferation makes it difficult to identify the most actionable and relevant next steps.

A recommended improvement is to run a consolidation and prioritization pass:

- **Review all existing idea files.**
- **Merge or discard duplicate and overlapping ideas** (e.g., keep only the most recent or comprehensive "consolidation", "curation", etc. file for each theme).
- **Select the 10 most interesting, actionable, and feasible ideas** to keep in the directory.
- **Remove all other idea files** to keep the directory focused and manageable.
- (Optionally) Maintain a summary index listing the current top 10 ideas.

This will streamline the project's "dream backlog", making it easier for contributors to focus on prioritized improvements and avoid duplication of effort.

## Feasibility Note
This is straightforward to implement and can be automated as part of a maintenance workflow or performed manually in a single session. It will immediately improve project clarity and focus for all contributors.
