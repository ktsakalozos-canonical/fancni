## Title
Nightly Dream Consolidation

## Description
The `misc/nightly-dreams` directory is currently overloaded with numerous idea files, including several with similar names and overlapping themes (e.g., multiple `consolidation` files, as well as files about prioritization, selection, curation, deduplication, etc.). To streamline future ideation and make past suggestions more actionable, introduce a nightly consolidation routine. This routine should:
- Review all idea files each night.
- Merge similar or duplicate ideas into a single, well-structured file.
- Retain only the 10 most relevant, actionable, and feasible ideas, deleting the rest.
- Tag consolidated ideas with metadata on their origin and rationale for selection or merging.
- Provide a summary of discarded ideas for traceability.

This would make the directory more manageable, help avoid idea overload, and encourage focused development.

## Feasibility Note
This is highly feasible and aligns with current development patterns. It will improve the project's organization and ensure that only the best ideas persist.
