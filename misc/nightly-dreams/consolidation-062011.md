Consolidate Nightly Dream Files

## Description
The `misc/nightly-dreams` directory currently contains a large number of files, many of which are variations on themes such as consolidation, curation, deduplication, prioritization, and other process improvements. This proliferation makes it difficult to quickly identify the most valuable and actionable ideas.

A next step would be to create a routine that periodically reviews these files, merges highly similar or redundant ideas, and curates the directory to contain only the 10 most relevant, distinctive, and feasible suggestions. This would streamline future development planning and prevent idea overload. The process could be automated or manual, but should include clear criteria for selection: feasibility, impact, distinctiveness, and alignment with project direction.
