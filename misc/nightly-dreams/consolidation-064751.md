# Nightly Dream: Consolidation of Nightly Dream Files

## Description

The `misc/nightly-dreams` directory currently contains a large number of individual idea files, many of which are variants of "consolidation," "curation," "deduplication," "prioritization," and similar themes, differentiated only by timestamps or unique IDs. This proliferation makes it harder to quickly review the active set of most relevant and actionable ideas.

**Proposal:**  
Introduce a regular (e.g., weekly) consolidation and pruning workflow:
- Review all files in `misc/nightly-dreams/`.
- For each theme with many timestamped variants (e.g., `consolidation-071407.md`, `consolidation-065230.md`, etc.), merge the most valuable insights into a single, up-to-date canonical file (e.g., `consolidation.md`).
- Discard outdated or redundant files.
- Keep only the 10 most interesting and feasible ideas as active files, as per the project's policy.

This will keep the idea pool focused, reduce noise, and make it easier to prioritize and act on the most promising suggestions.

## Feasibility

**High.**  
The process is largely organizational, requiring only periodic manual or automated file review and editing. Optionally, a simple script can automate the pruning and merging of files based on content similarity and timestamps. This consolidation will help maintain clarity, improve knowledge transfer, and support strategic planning.
